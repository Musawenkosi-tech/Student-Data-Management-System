*Student Data Management System

I built this Java app to help lecturers track student marks instead of messy Excel sheets. Works perfectly - add students, see reports, export to CSV that opens clean in Excel.

*What It Does ✅

1.Add students (ID, name, Undergrad/Grad, any # modules + marks)
2.Print nice report table with averages
3.Find high achievers (80%+ average)
4.Export CSV - opens perfect in Excel (no more merged cells!)
5.Save students so they stay after restart

*How To Run (Super Easy)

1. Clone repo 
2. Open in NetBeans
3. Run StudentApp.java
4. Type "access"
5. Use menu 1-6
   
*Menu Looks Like:

=== STUDENT MANAGEMENT SYSTEM ===
1. Add Student      2. Print Report    
3. High Achievers   4. Export to CSV
5. Save Students    6. Exit

*What I Used

1.Java (classes for Undergrad/Graduate students)
2.ArrayList (unlimited students)
3.File writing (CSV export + save data)
4.NetBeans

*What I Learned

1.Making classes that inherit (Undergrad vs Graduate)
2.Saving/loading files
3.Clean tables in console + Excel
4.Error handling so app doesn't crash

*Files It Makes

1.students.csv = Excel file
2.students.dat = my save file
